# Snake на Next.js

Мини-игра **Snake** на **Next.js (App Router)** + **TypeScript** + **Tailwind CSS**.
Проект готов к деплою на Vercel без дополнительной настройки.

## Запуск локально

```bash
npm install
npm run dev
```

Открыть в браузере: [http://localhost:3000](http://localhost:3000)

## Скрипты

- `npm run dev` — запуск dev-сервера
- `npm run build` — production-сборка
- `npm run start` — запуск production-сервера
- `npm run lint` — проверка линтером

## Управление

- `W` — движение вперёд относительно текущего направления
- `A` / `D` — поворот влево / вправо относительно текущего направления
- `↑ ↓ ← →` — классическое управление направлением
- `Space` — пауза / продолжить

> Разворот на 180° за один шаг запрещён.

## Игровая механика

- Поле: `24x24`
- Стартовая скорость: `150ms` на шаг
- Ускорение: каждые `5` очков скорость увеличивается
- Столкновение со стеной или телом змейки = `Game Over`
- Еда появляется только в пустой клетке
- `Best score` сохраняется в `localStorage`
